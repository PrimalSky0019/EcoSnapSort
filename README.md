ğŸ Getting Started
This project is a Next.js starter configured for Firebase Studio.

Clone the repository:

Bash

git clone https://github.com/your-username/eco-snap-sort.git
Install dependencies:

Bash

npm install
Configure Firebase: Add your Firebase configuration to your environment variables (.env.local).

Launch the app:

Bash

npm run dev
Open http://localhost:3000 to view the result. The main entry point is src/app/page.tsx.
